# 映画レコメンデーションアプリ (Movie Recommender)

このプロジェクトは、Flask、Pandas、Scikit-learnを使用したシンプルな映画レコメンデーションWebアプリケーションです。
ユーザーが選択した好きな映画に基づいて、類似のユーザーを探し（ユーザーベース協調フィルタリング）、そのユーザーが高評価をつけた他の映画を推薦します。

## 機能

- **トップ評価映画の表示**: アプリ起動時に評価の高い映画一覧を表示します。
- **映画レコメンデーション**: ユーザーが複数の映画を選択すると、AI（協調フィルタリング）がおすすめの映画を提案します。

## 必要要件

- Python 3.x
- 以下のPythonライブラリ（`requirements.txt`に含まれています）:
  - flask
  - pandas
  - scikit-learn

## インストールと実行方法

1. **リポジトリのクローンまたはダウンロード**
   プロジェクトファイルをローカル環境に保存します。

2. **依存ライブラリのインストール**
   ターミナルで以下のコマンドを実行し、必要なライブラリをインストールします。
   ```bash
   pip install -r requirements.txt
   ```

3. **アプリケーションの実行**
   以下のコマンドでWebサーバーを起動します。
   ```bash
   python app.py
   ```

4. **ブラウザでアクセス**
   ブラウザを開き、以下のURLにアクセスしてください。
   http://127.0.0.1:5000

## ファイル構成

- `app.py`: Flaskアプリケーションのメインファイル。ルーティングと画面表示を制御します。
- `recommender.py`: レコメンデーションロジックの中核。データの読み込み、類似度計算、推薦リストの生成を行います。
- `requirements.txt`: 必要なPythonパッケージのリスト。
- `templates/`: HTMLテンプレートファイル（`index.html`, `result.html`）が格納されているディレクトリです。
- `movies_100k.csv`, `ratings_100k.csv`: 映画データと評価データのCSVファイル（MovieLens Datasetの一部と思われます）。

## アルゴリズムについて

このアプリは**ユーザーベース協調フィルタリング**を使用しています。
1. ユーザーが選んだ映画に対して、仮想的に「自分」というユーザーを作成します。
2. 他のすべてのユーザーと「自分」との類似度（コサイン類似度）を計算します。
3. 自分と似た傾向を持つユーザー（類似ユーザー）を特定します。
4. 類似ユーザーが高く評価しているが、自分がまだ選んでいない映画を推薦候補として抽出します。
